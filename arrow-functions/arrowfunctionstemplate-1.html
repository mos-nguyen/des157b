Reverse Reverse

I got the idea of a function reversing an array from one of my old experiments from DES157A, and also looked through MDN to find the right syntax. It takes the given array and returns each item in reverse order, starting with the last item and going backwards to item 0.

Named Function

            function reverseSpell() {
                const magicSpell = ['this', 'is', 'a', 'magic', 'spell'];
                return magicSpell.reverse();
            }

            console.log(reverseSpell());

Function Expression

            const reverseSpell = function() {
                const magicSpell = ['this', 'is', 'a', 'magic', 'spell'];
                return magicSpell.reverse();
            }

            console.log(reverseSpell());

Arrow Function

            const reverseSpell = () => {
                const magicSpell = ['this', 'is', 'a', 'magic', 'spell'];
                return magicSpell.reverse();
            }

            console.log(reverseSpell());
